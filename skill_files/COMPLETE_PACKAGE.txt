💡 KEY TECHNIQUES
══════════════════════════════════════════════════════════════════════

✓ STORY FIRST, IDEAS EMBEDDED
  Ideas emerge from story, never interrupt it

✓ MULTI-LEVEL ACCESSIBILITY
  Level 1: Plot anyone can follow
  Level 2: Intellectual depth for smart readers  
  Level 3: Meta/symbolic layer for experts

✓ HIGH-LOW BALANCE
  Intellectual (complex ideas, ambiguity, literary prose)
  Accessible (clear plot, emotion, suspense)

✓ INVESTIGATION STRUCTURE
  Reader discovers alongside protagonist
  Both solving mystery together

✓ LEAN DESCRIPTION
  Precision over profusion
  Only meaningful details
  Eco-style discipline

✓ EMOTIONAL + INTELLECTUAL
  Both required for bestseller
  Balance ratio: 0.8:1 to 1.5:1
