## 🚀 How to Install & Use

### Installation
1. Download **book-writer.skill**
2. In Claude, go to Settings → Skills
3. Click "Import Skill"
4. Upload the .skill file
5. Enable the skill

### First Steps
Once installed, just start talking to Claude naturally:

**"I'm writing an Eco-style novel about [your topic]. I need to balance scientific accuracy with literary elegance for three audiences: general intelligent readers, scientists, and people interested in spiritual questions."**

Claude will automatically load the appropriate resources and adapt all guidance to your Eco-style requirements.

### Running Analysis Tools

The skill includes two Python scripts you can run on your manuscript:

```bash
# Basic statistics (word count, pacing, dialogue %)
python scripts/manuscript_stats.py your_manuscript.txt

# Advanced analysis (timeline, repetition, hooks, confusion)
python scripts/timeline_checker.py your_manuscript.txt
```

Then share the results with Claude for detailed interpretation and recommendations.

## 📊 What the Tools Reveal

### manuscript_stats.py Shows:
- Total words, sentences, paragraphs
- Reading time estimate
- Average sentence length (are you too wordy?)
- Dialogue percentage (too much exposition?)
- Chapter lengths (pacing issues?)
- Most common words (overused terms?)

### timeline_checker.py Shows:
- **Timeline**: Date consistency, duration logic
- **Repetition**: Overused words, sentence patterns, thematic vs wasteful
- **Terminology**: Inconsistent naming ("codex" vs "manuscript")
- **Chapter hooks**: Weak openings, flat endings
- **Confusion**: Contradictions, unclear pronouns, ambiguous passages

## 🎓 The Eco Philosophy

This skill embodies Umberto Eco's approach:

### Intelligence Over Entertainment
- Write for engaged, active readers
- Trust readers to do interpretive work
- Complexity is feature, not bug

### Precision Over Profusion
- Few meaningful details, not exhaustive description
- Every word earns its place
- Lean prose that rewards attention

### Ideas Through Story
- Philosophy emerges from character conflict
- Science discovered through investigation
- Reader participates in meaning-making

### Fair Play Mystery
- Reader has access to same clues
- Solution follows from established facts
- Difficult through volume/interpretation, not withholding
